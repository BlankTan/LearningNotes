# 二叉树的下一个节点

## 来源

https://www.nowcoder.com/practice/9023a0c988684a53960365b889ceaf5e

## 描述

给定一个二叉树和其中的一个结点，请找出中序遍历顺序的下一个结点并且返回。注意，树中的结点不仅包含左右子结点，同时包含指向父结点的指针。

## 步骤

- 寻找右子树，如果存在就一直找右子树的最左边就是下一个节点
- 没有右子树，就寻找它的父节点，一直找到它是父节点的左子树，打印父节点
